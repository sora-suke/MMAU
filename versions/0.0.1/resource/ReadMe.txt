--------このMODを利用するに当たって--------

･導入して、ワールドが壊れても責任は負えません。バックアップは定期的に取っておきましょう。
･クラッシュが起きたり不具合があれば他の導入MODやレポートも一緒に報告して欲しいです。(直せないかもしれませんが。)
･二次配布はDM等で許可を取ってからやってください。(MODパックも含む。)
･営利利用は基本禁止です。
･需要無いと思いますがソースが欲しければここやDMで言ってくれれば配布します。
･このMOD は クリエイティブ・コモンズ 表示 - 非営利 - 継承 4.0 国際 ライセンスの下に提供されています。


--------このMODは?--------
※現在、テスト版で要素が圧倒的に少なく、さらに仕様がコロコロ変わるかもしれません!
様々な素材(0.0.1現在2種類)と便利アイテムを追加します。
ネタアイテムや自分の趣味のネタがあるのでそういうのが嫌な人はオススメしません。


--------追加要素--------

----追加アイテム/ブロック----
ネギとその鉱石、圧縮ブロック、ツール一式
クリスタルとその鉱石、圧縮ブロック、ツール一式

----自然----
クリスタル鉱床

--------導入方法--------
前提:forge10.13.4.1448以降
このJarファイルをmodsに放り込む
